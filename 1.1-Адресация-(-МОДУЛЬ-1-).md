# HQ-CLI

`su root`

Сначала выключаем Network Manager

`systemctl stop NetworkManager`

`systemctl disable NetworkManager`

`systemctl mask NetworkManager`

После прописываем конфигурацию интерфейса ( `networking` перезапустить сможете только после настройки dhcp )
