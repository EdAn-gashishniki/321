# Правило

❗❗❗ПРОПИСЫВАЕМ ПРАВИЛО НА ISP, HQ-RTR и BR-RTR❗❗❗

Добавим правило для NAT в `nftables.conf`

`nano /etc/nftables.conf`

Указываем интерфейс, смотрящий в сторону Интернета

<img width="526" height="100" alt="image" src="https://github.com/user-attachments/assets/01cfbf5b-6eba-4de4-aa0e-46f580911a16" />


table inet nat {

        chain POSTROUTING {
        
        type nat hook postrouting priority srcnat;
        
        oifname "ens192" masquerade
        
        }
       
}


`systemctl enable nftables` - добавление службы в автозапуск

`systemctl start nftables` - включение службы
