# HQ-RTR

Устанавливаем chrony

apt install chrony

Удаляем конфигурации другой ntp утилиты

apt remove --purge systemd-timesyncd

Редактируем конфигурацию chrony, прописываем конфигурацию и комментируем подчёркнутую строку

nano /etc/chrony/chrony.conf

systemctl restart chronyd
